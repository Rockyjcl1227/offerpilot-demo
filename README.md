# OfferPilot

面向大学生求职季的 `求职申请管理看板` 交互式 MVP，用于美团测评题演示。

## 当前版本重点

- 保留以用户为中心的主界面结构
- 顶部增加 `用户端 / 管理端` 身份切换
- 首页强化“当前状态 + 下一步动作”
- 材料中心增加 AI 工作台与简历优化提示
- 引入浏览器内 `IndexedDB` 持久化用户、岗位和埋点事件

## 当前角色与数据

- 单用户：`林知许`
- 单管理者：`招聘产品经理`
- 数据存储在浏览器本地 `IndexedDB`

## 用户端链路

- 今日总览
- 申请看板
- 日历提醒
- 材料中心
- 岗位详情抽屉
- 新增申请

## 管理端链路

- 管理后台
- 用户链路转化
- 真实埋点事件流

## 已接入的真实埋点事件

- `view_dashboard`
- `view_page`
- `open_application_detail`
- `change_stage`
- `create_application`
- `view_materials`
- `switch_to_user`
- `switch_to_manager`

这些事件会写入本地数据库，可在管理后台里看到。

## 本地运行

直接打开 [index.html](./index.html) 即可，或在当前目录运行：

```powershell
python -m http.server 8090
```

然后访问 `http://localhost:8090`

## 演示建议

1. 先从用户端进入，看总览和看板
2. 打开某个岗位详情
3. 修改一个岗位状态
4. 新增一个岗位申请
5. 切到管理端查看埋点事件和链路变化
